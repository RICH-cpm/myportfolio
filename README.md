## что включено

Это готовый проект портфолио, содержащий:

### исходный код
- **index.html** - 580 строк HTML5
- **css/styles.css** - 670 строк CSS3
- **js/script.js** - 570 строк Vanilla JavaScript

### документация
- **README.md** - руководство (350+ строк)
- **QUICK_START.md** - старт (150+ строк)
- **DEVELOPMENT.md** - файл с подробностями

### структура проэкта
```
portfolio/
├── index.html              (Главный HTML)
├── css/
│   └── styles.css         (Все стили)
├── js/
│   └── script.js          (Вся интерактивность)
├── assets/
│   └── images/
│       └── achievements/  (Для сертификатов)
├── README.md              (Полная документация)
├── QUICK_START.md         (Быстрый старт)
└── .gitignore            (Git конфигурация)
```

## функциональность

### 1. Навигация
-  Sticky navbar с логотипом
-  6 основных секций (Home, About, Skills, Projects, Achievements, Contact)
-  Плавный скролл между секциями
-  Активная ссылка навигации (highlight)
-  Мобильное меню (hamburger)
-  Поддержка клавиатурной навигации (Alt+↑/↓)

### 2. Home Section
-  Героический баннер с приветствием
-  Две CTA кнопки (Projects, Contact)
-  Анимированный код-блок
-  Анимирующая стрелка для скролла
-  Фоновые градиентные эффекты

### 3. About Me Section
-  Короткая биография
-  Мотивация для обучения в России
-  Сетка из 8 технологий
-  Интерактивные карточки tech-stack

### 4. Skills Section
-  6 основных навыков с иконками
-  Progress bars с анимацией
-  Hover эффекты
-  Адаптивная сетка

### 5. Projects Section
-  4 примера проектов
-  Карточки с изображениями (SVG плейсхолдеры)
-  Описание, технологии, ссылка на GitHub
-  Hover эффекты с трансформацией

### 6. Achievements Section
-  8 плейсхолдеров сертификатов
-  Модальное окно для просмотра (lightbox)
-  Интерактивные карточки
-  SVG изображения для быстрой загрузки

### 7. Contact Section
-  3 способа связи (Email, GitHub, Telegram)
-  Интерактивные карточки контактов
-  Форма обратной связи с валидацией
-  Система уведомлений об отправке
